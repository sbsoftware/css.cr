## How To Add Missing CSS

1. Add property overloads in `src/stylesheet.cr` with `prop`, `prop2`, ... and include typed inputs (avoid raw `String`-only overloads).
2. For keyword-heavy properties, add `css_enum` values in `src/css/enums/*.cr` and use `CSS::Enums::*` types in property signatures.
3. For free-form values that should not be quoted, use `transform_string` to map `String` to a custom wrapper with `to_css_value`.
4. Add focused specs in `spec/*_spec.cr` for rendered CSS output and at least one typed path.
5. Regenerate this index: `crystal scripts/generate_coverage.cr`.
